FireStorageシステム情報管理

中古パソコン販売で威力を発揮するシステム情報の一元管理

なぜFireStorageが重要なのか

中古パソコン販売での課題

  • お客様からの詳細な問い合わせ対応（グラボの型番、液晶パネルの仕様など）
  • 販売後のドライバ提供やサポート対応
  • システム情報を毎回調べ直す手間

FireStorageの優位性

  • ログイン不要で即座にアップロード可能
  • どこからでもアクセス可能な情報管理
  • スマホからも確認・共有が可能

システム情報自動送信バッチ

@echo off
systeminfo > %TEMP%\sysinfo_%COMPUTERNAME%.txt
curl -F "file=@%TEMP%\sysinfo_%COMPUTERNAME%.txt" [FireStorageURL]
echo システム情報をFireStorageに送信しました
pause

テスト方法

  1. 1. 上記コードをメモ帳にコピー
  2. 2. [FireStorageURL]を実際のアップロードURLに変更
  3. 3. 「sysinfo_sender.bat」として保存
  4. 4. 右クリック→「管理者として実行」でテスト
  5. 5. FireStorageにファイルが送信されたか確認

パソコンセッティング業務の効率化

超戦略２で実践したパソコンセッティング業務で、複数アプリの同時起動によりセッティング時間を大幅短縮。この技術は他の繰り返し作業にも応用可能です。

@echo off
echo 出品作業環境を起動中...
start "" "C:\Program Files\Google\Chrome\Application\chrome.exe" "https://auctions.yahoo.co.jp"
timeout 3
start "" "https://drive.google.com"
timeout 2
start "" "C:\Program Files\速売り君\速売り君.exe"
echo 起動完了！出品作業を開始してください。
pause
